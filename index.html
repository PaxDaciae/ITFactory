Technoloy

Here are the languages I used for this project:

JavaScript

  1. Dinamic Background Gradient Opacity: this feature is based on a Scroll listener to dynamically change the opacity of the welcome screen background gradient.

  2. Color Theme change: changes the page's colors based on a locally-stored 'theme' variable. The changes are mediated though CSS variables, a series of classes, and (very rarely) inline styles.

  3. Local storage of Theme: changes the value of the locally-stored 'theme' variable based on the user's selection of theme.

  4. Stored Theme detection and default theme: the website checks if a Theme value is saved in local storage. If not (first-time access, local storage has been cleared, private browsing, etc), a default value, the Dark Theme, is saved to local storage and applied on the page.

  5. View shift on nav items: the nav items are simple <li> items. Viewport shift is mediated through a JS function that uses a simple if() conditional and retrieves an element's relative position using the .getBoundingClientRect() method -- or uses the 0 vertical position for the Welcome screen -- and then navigates to that value using the .scrollTo() metod on the <body> element.

  6. DOM manipulation: in this example, DOM manipulation is limited to retrieving elements and attribute (style, class, Id) manipulation.

HTML

  1. SEO: Search Engine Optimization of the site was performed through the use of semantic HTML tags, most importantly: <nav>, <section>, <article>, <footer> and <header>. Metadata was also used to promote SEO rating through the use of <title> <meta description=""> and <img alt=""> attributes.

  2. Accessibility - alt attribute some accesibility features were provided through the use of alt attributes on <img> tags.

  3. Scripts - Defer Attribute the defer attribute was used to ensure that JavaScript scripts are fully-loaded before the page is displayed to the user, in order to avoid janky behavior or errors.

  4. Fonts - Preload Attribute the preload attribute was applied to the Google Font link in order to ensure that the proper font is displayed on the page.

  5. Interactivity - Onclick attribute the onclick attribute was applied to HTML elements to allow their linking to various functions in JavaScript.

CSS

  1. Shifting Welcome Screen Background Gradient Position the color-shifting gradient background on the welcome screen was created fully-in CSS, using resizing and a @keyframes ruleset manipulating the gradient's position relative to the viewport.

  2. Parallax: this effect was created mostly in CSS, by defining contexts with different perspectives (using 'perspective' and 'transformY()' properties'). These contexts move at their own paces relative to the viewport, allowing elements placed in these context to appear to slide by each other at different speeds.

  3. @media Breakpoints: used to make the website responsive and good-looking on multiple screen sizes. The website was developed using a mobile-first approach.

  4. @keyframes Animations: they provide visual interest to the user.

  5. Variables: used to change the properties inside multiple selectors easily, at once. Mostly used for colors in this site.

  6. Alignment using Flexbox: used to align items vertically and/or horizontally in order to preserve responsiveness and looks.

  7. Auxiliary Classes: used to quickly and easily apply / remove / toggle different style properties to multiple HTML elements.
